# spunn

Stock price prediction by using Recursive Neural Networks!

## Dependencies
yahoo_fin
pandas
numpy

## Installation
```bash
conda create --name spunn python=3.10
pip install yahoo_fin
```
```bash
conda install -c anaconda ipykernel
python -m ipykernel install --user --name=spunn
```

```bash
$ pip install spunn
```
